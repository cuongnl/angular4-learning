ATTRIBUTE DIRECTIVE: thay đổi cách hiển thị (style) hoặc hành vi của một DOM element/Component/Directive khác

------------------CLASS BINDING------------------

Lý thuyết
- Cần thay đổi (thêm, xóa) một số class tùy thuộc vào một số điều kiện nào đó
- Là câu điều kiện
- Mẫu câu: *ngIf="expression"
Coding
Demo
name: {{user.name}}
age: {{user.age}}
Năm nay tuổi bạn là chẵm
Năm nay tuổi bạn là lẻ

------------------CẤU TRÚC NGFOROF------------------

Lý thuyết
- là 1 property đặc biệt
- Là duyệt array trên template
- Mẫu câu: *ngFor="let item of array"
- Thêm, xóa, thay đổi cấu trúc cây DOM
Coding
Demo
({{idx}})/({{total}}): {{author.id}} - {{author.firstName}} {{author.lastName}}
